# GENR-DEP-JSON

> Generate json file of pre-installed dependencies(node_modules) with their exact version.


## Install

```bash
npm i -g genr-dep-json
```

## Usage

1. Install package as global module
2. Goto root folder of your project and open command prompt
3. Run ```generate-json```
